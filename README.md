# Dashboard

A web-based dashboard application built using **Vue**, **TypeScript**, **HTML**, **JavaScript**, and **CSS**. This project serves as an interactive and user-friendly interface for managing data and visualizations.

## Features

- **Vue.js Framework**: Leverages the power of Vue.js for a reactive and component-based UI.
- **TypeScript**: Ensures type safety and scalable development.
- **Responsive Design**: Optimized for various devices, from desktops to mobile phones.
- **Dynamic Visualizations**: Supports interactive charts and data visualizations.
- **Customizable Components**: Easily extendable for specific use cases.

## Getting Started

Follow these instructions to set up and run the project locally.

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or later)
- [npm](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mgrl39/dashboard.git
   cd dashboard
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server:
   ```bash
   npm run serve
   # or
   yarn serve
   ```

4. Open your browser and navigate to `http://localhost:8080`.

## Project Structure

- `src/`: Contains the main application code.
  - `components/`: Reusable Vue components.
  - `views/`: Page-level components representing different routes.
  - `assets/`: Static resources like images and styles.
- `public/`: Static files served directly.
- `package.json`: Project configuration and dependencies.
